Welcome

This is an app that let you analyze the performance of various hashing collision handling methods:

  • Linear Probing collision handling
  • Quadratic Probing collision handling
  • Linear Quotient collision handling

To use the app, please follow these steps:

  1. Choose a set of keys to be search in the hash table
  2. Choose a collision resolution method
  3. Choose a packing density
  4. Press the search button

Note that:

  • The hash table is already created when you launch this program
  • The hash table uses random alphanumeric keys. The data can be found in the data folder
  • All the text display of this program can be modified and access through the templates folder
